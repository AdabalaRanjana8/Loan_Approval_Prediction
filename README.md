# Loan_Approval_Prediction
Loan Approval Prediction including data preprocessing, SMOTE, Logistic Regression, and Random Forest.

## Project Overview

This project predicts whether a loan application will be approved or rejected using Machine Learning techniques.
The goal is to help banks and financial institutions make faster and more accurate loan approval decisions.

## Dataset
The dataset contains information about loan applicants such as:

* Gender
* Married Status
* Dependents
* Education
* Self Employment
* Applicant Income
* Coapplicant Income
* Loan Amount
* Loan Amount Term
* Credit History
* Property Area
* Loan Status

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Imbalanced-learn (SMOTE)
* Jupyter Notebook
* VS Code

## Project Steps

1. Data Loading
2. Data Preprocessing
3. Missing Value Handling
4. Label Encoding
5. Feature Scaling
6. Handling Class Imbalance using SMOTE
7. Model Training
8. Model Evaluation
9. Business Interpretation

## Models Used

* Logistic Regression
* Random Forest Classifier

## Conclusion

A Machine Learning model was successfully developed to predict loan approval status. Logistic Regression achieved the best performance and is recommended for deployment.
